# OSINT Tracker - Devin AI 项目简介

## 🎯 项目概述

OSINT Tracker 是一个开源情报收集和分析平台，专门用于：
- 📧 邮箱查询和数据泄露检测
- 📱 电话号码查询（美国、印尼）
- 🌐 社交媒体账户查询
- 🗺️ 地理位置可视化
- 👥 用户管理和权限控制
- 💰 积分系统

## 🏗️ 技术架构

### 后端 (Python)
- **框架**: FastAPI
- **数据库**: SQLite (默认) / MongoDB
- **缓存**: Redis
- **认证**: JWT + bcrypt

### 前端 (React)
- **框架**: React 18
- **UI**: Tailwind CSS + Radix UI
- **地图**: Leaflet + Mapbox
- **状态管理**: React Hook Form
- **构建**: Create React App + Craco

## 🔌 外部集成

### 核心 API 服务
- **RapidAPI** - 多个服务集成
- **OSINT Industries** - 专业 OSINT 数据
- **Have I Been Pwned** - 数据泄露检查

### 电话查询服务
- **Truecaller** - 全球电话号码查询
- **IPQualityScore** - 电话验证和风险评估
- **AceLogic** - 电话号码情报
- **Melissa** - 电话验证服务

### 社交媒体服务
- **WhatsApp** - WhatsApp 账户查询
- **Telegram** - Telegram 用户查询
- **Facebook** - Facebook 档案查询

## 🚀 核心功能

### 1. 邮箱情报收集
- 数据泄露历史查询
- 邮箱关联账户发现
- 社交媒体档案匹配

### 2. 电话号码分析
- 运营商信息查询
- 地理位置定位
- 社交媒体关联
- 风险评估分析

### 3. 社交媒体调查
- 跨平台账户关联
- 档案信息提取
- 活动轨迹分析

### 4. 数据可视化
- 交互式地图展示
- 关系网络图谱
- 时间线分析
- 统计报表
